Why a blog?

"If people cannot write well, they cannot think well. And if they cannot think well, others will do their thinking for them." - George Orwell

I have been keeping notes in my phone about stuff that fascinates me for a long time, but I would never really try to think them through. I'd just forget about them until I'd add another idea to the list. This is where I try to mold those notes into something readable.

How this website was made

I made this website using Jekyll. It is hosted for free on Github Pages. Jekyll is a static site generator that's very easy to get the hang of if you know some basic HTML and CSS. Github Pages is a Github service that allows users to host static websites for free. Both services have excellent documentation!
