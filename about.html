Welkom op Weetgraag, een blog over technologie, geneeskunde en wetenschap. Hier diep ik mijn interesses uit. Ondertussen studeer ik geneeskunde (interesse: psychiatrie), en leer ik mezelf programmeren. Ik ben groot voorstander van een pseudoniem internet. Stuur me ouderwets een mailtje!

Waarom in het Nederlands?

Door mijn grotendeels Engelstalige mediadieet gaat mijn Nederlandse vocabulaire achteruit. Dat vind ik zonde. Wij Nederlanders mogen best wat tegenwicht bieden aan de opkomst van dat Angelsaksisch gebrabbel.

Hoe is deze website gemaakt?

Deze website gebruikt Jekyll, een "static site generator" die makkelijk te leren is met een beetje voorkennis van HTML en CSS. Hosting is gratis via Github Pages. Alleen de domeinnaam kost dus een paar euro per jaar!
