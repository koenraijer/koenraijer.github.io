Programming

Programming wasn’t something I had anticipated learning in 2020. Not a surprise, considering no one anticipated anything in 2020. My travels to both Indonesia and India were scrapped off the agenda, so I tried finding a research project to help with. That ended up not happening either, so I decided to use the time I’d taken off for research to learn some programming!

What's the point?

  • Technology will play an undeniably big role in the future of healthcare.

  • Coding ability gives new perspective to problem-solving.

  • Develop your own ideas!

I first wrote a simple tool to help me rebalance my stocks in Python. It's simple and ugly, but it works! I turned it into a web application using a Python framework called Flask, and then I realized it was going to be an expensive undertaking to host it... So I abandoned Python and started focussing on HTML, CSS and JS to build this website.

Data analysis

Then I got interested in the possibilities of doing data analysis. I picked R for its popularity in academia. R is open-source software that doesn't have a user interface like SPSS does; it's a programming language that works by typing commands in a console. Its active userbase maintains and releases a large amount of packages that do anything you want. There's a package for everything. That's what makes it so versatile. I got the hang of a popular plotting package called ggplot2 pretty quicky, using free datasets that R comes equipped with.

Why it's important

  • The value of science in applied fields hinges on its ability to improve human behaviour.

  • We have more data than ever. It's all worthless without meaningful visualization.

  • The world is becoming increasingly confusing. We're in need of clarity.

What's next?

  • Complete Harvard's CS50 Introduction to Computer Science. Completing this course will give me a sturdy theoretical basis for any future coding endeavour.

  • Get proficient in data analysis using both Python and R. Get good at interactive data visualization; data exploration, manipulation and analysis.

  • Develop my skills in mathematics. Mathematics are a fundemental tool in programming.
